# Logo - Partiák

## Soubory

- `favicon.svg` - Ikona pro favicon (64x64)
- `logo.svg` - Velké logo pro potenciální použití
- Pro Apple Touch Icon (.png) můžete použít online konvertor:

## Vytvoření apple-touch-icon.png

1. Otevřete `favicon.svg` v prohlížeči
2. Použijte online nástroj jako:
   - https://cloudconvert.com/svg-to-png
   - https://convertio.co/svg-png/
3. Nastavte velikost na 180x180px
4. Stáhněte jako `apple-touch-icon.png`
5. Umístěte do root složky webu

## Design loga

**Ikona:**
- Zlatá čára (#C9A227) - Představuje cestu růstu a kariérního postupu
- Zelený bod (#00E676) - Výchozí bod, sportovní základ
- Šipka - Směr a cíl

**Použití:**
- Navigační logo: SVG ikona + text "Partiák"
- Favicon: Pouze ikona bez textu
- Všechny HTML stránky mají logo konzistentně

**Barvy:**
- Primární: #0E1B2E (tmavě modrá)
- Akcent: #C9A227 (zlatá)
- Success: #00E676 (zelená)
